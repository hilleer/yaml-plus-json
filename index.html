YAML Plus JSON

Effortlessly convert between YAML and JSON in your editor

YAML Plus JSON is an editor extension that makes it simple to convert files and selections back and forth between YAML and JSON — right inside your editor, without leaving your workflow.

Key features include:

  • Convert entire files or selected text from YAML → JSON
  • Convert entire files or selected text from JSON → YAML
  • Works directly from the command palette and explorer context menu
  • Lightweight, fast, and easy to use
