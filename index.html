YAML Plus JSON

Effortlessly convert between YAML and JSON in your editor

See It in Action

Converting between YAML and JSON using the command palette

YAML Plus JSON is an editor extension that makes it simple to convert files and selections back and forth between YAML and JSON — right inside your editor, without leaving your workflow.

Key features include:

  • Convert entire files or selected text from YAML → JSON
  • Convert entire files or selected text from JSON → YAML
  • Works directly from the command palette and explorer context menu
  • Lightweight, fast, and easy to use

Currently available for Visual Studio Code, with support for more editors planned. Note that available features may vary between editors.

Install for VS Code

What You Can Do

Rename a file and the content converts automatically

Rename config.yaml to config.json and the extension rewrites the file contents to match — no manual conversion needed.

  1. Right-click a .yaml or .json file in the Explorer
  2. Rename it with the opposite extension
  3. The file content is automatically converted

Comments survive the round-trip

YAML comments (# my comment) are preserved when converting to JSONC as line comments (// my comment), so you don't lose context switching formats.

  1. Open a .yaml file with comments
  2. Run Ctrl+Shift+P → Convert to JSON
  3. Comments appear as // … in the resulting .jsonc file

Convert an entire folder at once

No need to open files one by one — right-click any directory and convert all YAML or JSON files inside it in a single action.

  1. Right-click a folder in the Explorer panel
  2. Select Convert YAML files to JSON (or vice versa)
  3. All matching files in the directory are converted

And more — feature availability varies by editor. See the full README for a complete list of commands and options.
